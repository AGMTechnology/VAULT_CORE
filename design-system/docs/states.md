# States

## Interaction states
- Buttons: `default`, `hover`, `focus`, `disabled`
- Inputs: `default`, `focus`, `error`, `success`, `disabled`
- Toggle and checkbox: `checked`, `unchecked`, `disabled`

## Semantic states
- Feedback: `success`, `warning`, `error`, `info`
- Workflow: `pending`, `running`, `blocked`, `validated`, `failed`
- Agent: `idle`, `active`, `processing`, `error`, `complete`
- Gate: `pending`, `review`, `passed`, `failed`, `skipped`

## Domain states
- Contract lifecycle: `intake`, `qualification`, `enrichment`, `validation`, `publication`
- Run lifecycle: `pending`, `running`, `completed`, `failed`
